Colors

All color values are respresented and to be used with the accompanied hex number. This color palette was designed with accessibility in mind and careful consideration of contrast ratios.

To be used for accent text, h1, and h2 headings.

Oatmilk
#FFF5E8

To be used for webpage background, accent borders, paragraphs, h3, h4 and h5 headings over accent and main backgrounds.

Midnight
#000000

To be used for paragraph and other miscellaneous text not over the main background.
